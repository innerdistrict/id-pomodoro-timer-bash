# id-pomodoro-timer-bash
A simple terminal pomodoro timer written in Bash.

Usage: ./id-pomodoro_timer.sh [OPTIONS]

  -w, --work <minutes>        Work duration in minutes (default: 25)

  -b, --break <minutes>       Short break duration in minutes (default: 5)

  -l, --long-break <minutes>  Long break duration in minutes (default: 15)

  -h, --help                  Show this help message
